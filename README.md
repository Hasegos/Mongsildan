## 주요 기능

### 검색 기능

> 각 영상마다  태그, 영상제목, 채널명에 대해 같거나 비슷할 경우 검색가능하게 구현했습니다.

### 댓글 기능

> 댓글을 작성했을시 작성 시간, 댓글 작성자가 영상 주인으로 구현했습니다.

### 추천영상 

> 어휘간 유사도 분석 API를 사용하여 가장 밀접한 유사도를 가진 영상들이 추천영상으로 뜨게 구현했습니다.
